## 2) ‚õèÔ∏è Data Mining Projects

During my PhD, I published **two papers** on the discovery of **frequent itemsets** and **association rules**.  
These works rely on specialized tree structures that enable efficient recursion over items:  
- a **prefix tree** using the **PrefRec** algorithm,  
- a **suffix tree** using the **SufRec** algorithm.  

The **SufRec** algorithm proved to be **significantly more efficient**, and my implementation in C/C++ is both **more refined** and **more effective** than **PrefRec**, making it a strong choice compared to other existing algorithms.
